@whale Here's what I'm thinking so far for the style guide. Since there are a lot of people collaborating on this project it'll be helpful to have the code for each element in the guide itself. @dustintheweb specified we'll be using SCSS (fuck yeah), so it'll be super easy to just set up variables for all these colors/sizes/etc. But yeah, clicking the little "</>" icon will reveal the HTML/SCSS code for that element, and you'll be able to copy that info. The guide will also use the Bootstrap grid, and should be a living entity on the RGE website, much like Salesforce's styleguide (http://sfdc-styleguide.herokuapp.com/). Still working on components and the grid for now, let me know what you guys think so far.
